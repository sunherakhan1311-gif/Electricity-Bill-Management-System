# Electricity-Bill-Management-System

## Project Description
The Electricity Bill Management System is a Java-based application developed using JDBC and MySQL for efficient management of customer electricity billing records.

## Features
- Add customer records
- Automatic bill calculation
- Store customer details in MySQL database
- JDBC connectivity
- Exception handling

## Software Requirements
- Java JDK 21
- MySQL
- JDBC Driver
- VS Code

## Steps to Run the Project
1. Install Java JDK 21.
2. Install MySQL.
3. Create the database and table.
4. Configure JDBC connection.
5. Compile and run the Java program.
